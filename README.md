# Ранжування сайтів (PageRank)
Комп'ютерний проєкт з дискретної математики

Python-проєкт, який обчислює **PageRank** для вершин орієнтованого графа.  
Кожна вершина графа відповідає одному сайту, а орієнтоване ребро `A -> B` означає, що сайт A посилається на сайт B.

## Ідея проєкту

Мета - реалізувати простий варіант алгоритму **PageRank**, який:

- на вхід отримує файл з описом орієнтованого графа (список вершин та ребер);
- будує внутрішнє подання графа;
- ітеративно обчислює значення PageRank для кожного сайту;
- виводить словник `{"назва_сайту": значення_PageRank}`.

## Використаний алгоритм

Для обчислення рангу використовується класична ітераційна формула PageRank:

$$
PR(i) = \frac{1 - d}{N} + d \left( \sum_{j \in In(i)} \frac{PR(j)}{L(j)} + \frac{S}{N} \right)
$$

де:
- \( PR(i) \) — рейтинг вершини (сайту) *i*,
- \( d = 0.85 \) — коефіцієнт згасання (*damping factor*),
- \( N \) — кількість вершин у графі,
- \( In(i) \) — множина вершин, що мають посилання на *i*,
- \( L(j) \) — кількість вихідних посилань із вершини *j*,
- \( S \) — сума рангів **висячих вершин** (тих, що не мають вихідних ребер).

Ітерації виконуються доти, доки зміна PageRank для всіх вершин не стане меншою за `1e-6`.


## Формат вхідних даних

Вхідний файл має містити опис орієнтованого графа у спрощеному форматі DOT.  
Приклад:

```dot
digraph G {
    A -> B
    A -> C
    B -> C
    C
    D -> A
}
